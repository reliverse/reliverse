Reliverse is an open-source collection of developer tools and libraries at [reliverse.org](https://reliverse.org). Inspired by ecosystems like [oxc](https://github.com/oxc-project/oxc) and [UnJS](https://unjs.io), it provides a modular, Bun-first monorepo.

**Calm by default. Powerful by design.**

## A Developer Tools Ecosystem

Reliverse is designed for developers who want:

- **Modular libs and CLIs**: `@reliverse/*` packages for common tasks.
- **Bun-first tooling**: Fast runtime, package manager, and tests.
- **Transparency**: System actions are visible and reversible; configuration is explicit.
- **Accessibility**: Usable by default, for everyone.
