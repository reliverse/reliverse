**Why are Reliverse projects hosted under the Reliverse GitHub organization?**

Reliverse is developed and maintained by the **Reliverse** GitHub organization.

Reliverse is the technical and research layer behind Reliverse.

It provides the infrastructure, tools, and experimental technologies that power the Reliverse ecosystem.

In simple terms:

- **Reliverse** is the developer tools ecosystem and monorepo.
- **Reliverse** is the GitHub organization building and maintaining it.

Some projects under Reliverse may explore independent ideas, research directions, or experimental technologies: but Reliverse remains one of its core initiatives.

## Philosophy

Reliverse focuses on building developer tools and libraries: CLIs, runtimes, and modular packages that enable productive development. Inspired by ecosystems like oxc and UnJS, it prioritizes clarity, type safety, and Bun-first tooling.

Reliverse is the primary monorepo where these tools are developed and integrated.
