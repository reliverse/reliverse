# `@reliverse/tools-rse-plugin`

Utility plugin for the `rse` CLI.

## Purpose

This plugin contributes general-purpose tooling commands to the Reliverse developer CLI.

Current command families include:

- `rse escape`

## Notes

- Built on the Rempts file-based plugin model
- Uses `--overwrite` when replacing existing generated outputs is intentional
- Designed to stay automation-friendly and explicit
