# `@repo/convex`

Shared Convex package for Reliverse.

## Purpose

Contains Convex-facing code, generated API bindings, and shared backend logic for Convex-based flows.

## Scripts

```bash
bun run dev
bun run build
bun run typecheck
bun run init
```

## Notes

- Exposes generated Convex modules
- Depends on `@repo/env`
- Intended as a reusable backend package inside the monorepo
