# `@reliverse/sdk`

Shared SDK package for Reliverse.

## Purpose

Provides a typed SDK entrypoint for consumers that integrate with Reliverse APIs or shared contracts.

## Scripts

```bash
bun run build
bun run check
bun run typecheck
```

## Notes

- Export surface starts at `src/index.ts`
- Bun-first package
- Intended for external or cross-workspace consumption
