# `@repo/server`

Shared server runtime package for Reliverse.

## Purpose

Provides reusable server-side contracts, transport helpers, and backend composition utilities.

## Depends on

- `@repo/auth`
- `@repo/db`
- `@repo/env`
- `@orpc/*`
- `elysia`

## Notes

- Workspace package
- TypeScript-first
- Intended to be consumed by server apps such as `apps/api`
