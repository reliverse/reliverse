# `@repo/reliverse-wiki`

Docs and wiki app for Reliverse.

## Purpose

This app hosts the documentation surface, guides, and knowledge pages for the Reliverse ecosystem.

## Stack

- Next.js
- Fumadocs
- MDX-based content flows

## Scripts

Run from the monorepo root:

```bash
bun run dev:docs
```

## Notes

- Intended for product docs, developer guides, and reference material
- Prefer documenting stable architecture and workflows here rather than scattering long-form docs across package internals
